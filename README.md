# Презентація "Про мене"

## Опис
Це шаблон презентації у LaTeX (Beamer) на тему "Про мене".  
Студенти повинні використати шаблон для практичної роботи №6 з інформатики.

---

## Інструкція

1. Перейдіть за посиланням на шаблон у **GitHub Classroom**.  
2. Скопіюйте або клонувати репозиторій у своє середовище (Overleaf або TeXmaker).  
3. Відкрийте файл шаблону **`presentation.tex`**.  
4. Замініть текст на інформацію про себе:  
   - ім’я, курс, спеціальність, основні предмети  
   - короткий вступ  
   - академічні досягнення та проєкти  
5. Додайте слайди:  
   - **Навчальні інтереси** – улюблені предмети, напрямки навчання, інструменти  
   - **Особисті навички** – комунікація, робота в команді, креативність  
   - **Хобі та захоплення** – спорт, мистецтво, музика, малювання  
   - **Плани на майбутнє** – цілі в навчанні та професійній діяльності  
6. Збережіть зміни і скомпілюйте файл у PDF.  
7. Завантажте **`.tex` та PDF** назад у свій репозиторій GitHub Classroom.
